KI: HCP Consul - No support for gossip key rotation

Documented at: 10/06/2022 Review by: 31/01/2023 The HCP Consul platform does not permit operators to execute a Gossip Key rotation. The Gossip key is used to encrypt the consul agent gossip communication.
